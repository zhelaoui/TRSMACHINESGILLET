# V1_SCOPE

## Inclus dans la V1 front

- Interface de navigation complète entre les pages métier principales.
- Affichage de données mockées cohérentes pour une machine unique.
- Composants visuels réutilisables (cartes KPI, panneaux, planning, tableaux).
- Structure de dépôt propre et prête à évoluer.

## Explicitement exclu

- Backend / API / logique serveur.
- PHP et MySQL.
- Authentification et gestion utilisateurs.
- Calculs TRS côté serveur ou persistance de données.
