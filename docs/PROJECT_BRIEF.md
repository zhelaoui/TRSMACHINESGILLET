# Project Brief

## Vision

Construire une nouvelle application web TRS pour Gillet à partir d'une base propre.

La première version doit être simple, lisible et facilement utilisable sur le terrain.

## Pourquoi repartir de zéro

L'objectif est d'éviter un projet difficile à maintenir ou une logique de calcul confuse.
La base doit être claire dès le départ afin de pouvoir faire évoluer ensuite le produit de manière fiable.

## Périmètre V1

La V1 se limite à :
- une seule machine
- un planning d'affaires
- des horaires d'ouverture
- un historique d'état marche / arrêt
- des maintenances
- des non-conformités
- le calcul des temps et du TRS V1
- une interface web simple

## Utilisateurs

### Opérateur
- consulte le planning de la machine
- déclare une maintenance
- déclare une non-conformité

### Responsable / superviseur
- paramètre la machine
- définit les horaires d'ouverture
- planifie les affaires
- consulte les indicateurs

## Résultat attendu

Obtenir une première application fiable qui permette de démontrer la valeur du suivi TRS basé sur le temps, avant d'ajouter des fonctions plus avancées dans de futures versions.
