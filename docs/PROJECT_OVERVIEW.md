# PROJECT_OVERVIEW

TRSMACHINESGILLET est une application de suivi TRS simplifié pour la société Gillet.

## Cible métier V1

- Démarrage avec **une seule machine**.
- Modèle TRS basé sur les **temps** (ouverture, requis, fonctionnement, théorique, utile).
- Mise à disposition d'une interface claire pour visualiser machine, planning, événements et indicateurs.

## Orientation technique de l'étape actuelle

- Front-end statique uniquement.
- Données fictives/mockées.
- Aucune logique serveur ou stockage persistant.
